=======================================

### Running tests

Install dependencies:
```
npm i
```

Run ganache:
```
npm run ganache
```

While ganache is running, run tests:
```
npm test
```

*Due to use of ganache’s `evm_increaseTime`, you should restart ganache after each run of tests.*
